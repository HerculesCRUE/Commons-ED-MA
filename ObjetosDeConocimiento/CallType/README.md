# Hércules ED. Objeto de conocimiento CallType

La entidad eroh:CallType (ver Figura 1) representa el tipo de convocatoria en el Curriculum Vitae en la plataforma Hércules. Dispone de varias opciones:
- Competitivo
- No competitivo
- Otros
