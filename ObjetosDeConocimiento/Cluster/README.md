# Hércules ED. Objeto de conocimiento Cluster

La entidad eroh:Cluster (ver Figura 1) representa un cluster de investigadores.

Las propiedades son las siguientes:

- roh:title
- vivo:description
- dct:issued
- roh:hasKnowledgeArea
- eroh:clusterPerfil
- rdf:member
