# Hércules ED. Objeto de conocimiento EventType

La entidad eroh:EventType (ver Figura 1) representa el tipo de evento en el Curriculum Vitae en la plataforma Hércules. Dispone de varias opciones:
- Congreso
- Jornada
- Seminario
- Otros
